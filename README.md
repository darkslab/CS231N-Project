### CS231N Project ###

A Stanford CS231N course project.


#### System Requirements ####

*   Python 3.8.7


#### Installation ####

*   `pip install -r requirements.txt`
*   `cd dependencies/`
*   `unzip baselines-master.zip`
*   `cd baselines-master`
*   `python setup.py install`
*   `cd .. && rm -rf baselines-master/ && cd ..`
